# ✈️ Flight Ticket Management System

Dự án Quản lý bán vé máy bay, được phát triển theo mô hình Agile với kiến trúc Backend và Frontend hoàn toàn tách biệt.

## 🛠 Công nghệ & Ngôn ngữ sử dụng (Tech Stack)

**Backend (RESTful API):**
* **Ngôn ngữ:** Java 17
* **Framework:** Spring Boot 3.x
* **Cơ sở dữ liệu:** MySQL
